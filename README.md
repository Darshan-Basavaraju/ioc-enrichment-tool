# IOC Enrichment Tool

A Python tool that queries the VirusTotal API to enrich IOCs for threat intelligence workflows.

## Features
- Single domain reputation lookup
- Extracts malicious verdict counts, vendor detections, and reputation scores
- Identifies which vendors flagged a domain as malicious

## Usage
python3 vt_lookup.py google.com

## Tools
- Python 3
- VirusTotal API v3
